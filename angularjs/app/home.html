MEAN Stack Development

This app is an example for MEAN stack development.This hello world app uses the below techology stack -

  • Angular 1.6.9
  • Bootstrap 4.0
  • MongoDB 3.6
  • Express JS 4.16

This is an effort by me to understand the below topics w.r.t MEAN stack techologies -

  • A fully functional end to end integration with MEAN stack.
  • Custom Angular Directives
  • CRUD operations with Mongo
  • REST services through ngResource
  • Error handling
  • Working with Modals
